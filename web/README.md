# وب اپلیکیشن دانلود فیلم و سریال

یک وب اپلیکیشن حرفه‌ای و مدرن برای نمایش و دانلود فیلم و سریال با رابط کاربری زیبا و واکنش‌گرا.

## ویژگی‌ها

✅ **احراز هویت با توکن تلگرام** - ورود با شناسه کاربری تلگرام  
✅ **نمایش فیلم/سریال‌ها** - نمایش با پوستر و اطلاعات کامل  
✅ **جستجوی پیشرفته** - جستجو بر اساس نام، ژانر، سال و نوع محتوا  
✅ **پروفایل کاربری** - نمایش آمار و تاریخچه بازدیدها  
✅ **پشتیبانی از سریال** - نمایش فصل‌ها و قسمت‌ها با کیفیت‌های مختلف  
✅ **طراحی مدرن** - استفاده از Tailwind CSS و Glass Morphism  
✅ **واکنش‌گرا** - سازگار با موبایل، تبلت و دسکتاپ  

## ساختار فایل‌ها

```
web/
├── config.php              # تنظیمات و اتصال به دیتابیس
├── login.php               # صفحه ورود
├── logout.php              # خروج از حساب
├── index.php               # صفحه اصلی
├── movie.php               # صفحه جزئیات فیلم/سریال
├── season.php              # صفحه فصل‌های سریال
├── episode.php             # صفحه قسمت‌های سریال
├── search.php              # صفحه جستجو
├── profile.php             # صفحه پروفایل کاربر
├── category.php            # صفحه دسته‌بندی‌ها
├── includes/
│   ├── header.php          # هدر مشترک
│   └── footer.php          # فوتر مشترک
└── .htaccess               # تنظیمات امنیتی
```

## نحوه استفاده

### 1. دسترسی به وب اپ

برای دسترسی به وب اپ، به آدرس زیر مراجعه کنید:
```
https://your-domain.com/web/
```

### 2. ورود به حساب کاربری

1. به صفحه ورود (`login.php`) بروید
2. شناسه عددی خود را از ربات تلگرام دریافت کنید
3. شناسه را در فرم ورود وارد کنید
4. پس از ورود موفق، به صفحه اصلی هدایت می‌شوید

**نکته:** کاربر باید قبلاً از طریق ربات تلگرام ثبت‌نام کرده باشد.

### 3. استفاده از وب اپ

- **صفحه اصلی:** نمایش دسته‌بندی‌ها، محبوب‌ترین‌ها و جدیدترین‌ها
- **جستجو:** جستجوی پیشرفته با فیلترهای مختلف
- **پروفایل:** مشاهده آمار و تاریخچه بازدیدها
- **جزئیات فیلم/سریال:** مشاهده اطلاعات کامل و لینک‌های دانلود

## ویژگی‌های فنی

### تکنولوژی‌های استفاده شده

- **PHP 7.4+** - Backend
- **MySQL/PDO** - دیتابیس
- **Tailwind CSS** - استایل‌دهی
- **Font Awesome** - آیکون‌ها
- **Vazirmatn Font** - فونت فارسی

### امنیت

- استفاده از Prepared Statements برای جلوگیری از SQL Injection
- Sanitize کردن ورودی‌های کاربر
- Session Management برای احراز هویت
- محافظت از فایل‌های حساس با `.htaccess`

### بهینه‌سازی

- استفاده از Glass Morphism برای UI مدرن
- Lazy Loading برای تصاویر
- Responsive Design برای تمام دستگاه‌ها
- بهینه‌سازی Query‌های دیتابیس

## تنظیمات

### تغییر آدرس سایت

در فایل `config.php`، آدرس `BASEURI` را تغییر دهید:
```php
define('BASEURI', 'https://your-domain.com/8');
```

### تغییر نام سایت

در فایل `config.php`:
```php
define('SITE_NAME', 'نام سایت شما');
```
